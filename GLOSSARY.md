# 게임 용어 정리

TFT 스타일 오토배틀러 게임의 용어 사전

---

## 기본 용어

| 용어 | 영어 | 설명 |
|------|------|------|
| **유닛** | Unit | 보드에 배치할 수 있는 캐릭터 |
| **기물** | Unit/Piece | 유닛의 다른 표현 |
| **보드** | Board | 4x4 그리드, 유닛을 배치하는 전투 영역 |
| **벤치** | Bench | 8슬롯 대기석, 배치 전 유닛 보관 |
| **상점** | Shop | 유닛 구매 장소 |
| **리롤** | Reroll | 2골드로 상점 갱신 |

---

## 유닛 스탯

| 용어 | 영어 | 설명 |
|------|------|------|
| **체력** | HP | 유닛의 생명력 |
| **공격력** | Attack | 기본 공격 피해량 |
| **공격속도** | Attack Speed | 초당 공격 횟수 |
| **사거리** | Range | 공격 가능 거리 (1=근접, 2+=원거리) |
| **방어력** | Defense | 받는 피해 감소량 |
| **마나** | Mana | 스킬 사용에 필요한 자원 |
| **이동속도** | Move Speed | 전투 중 이동 빠르기 |

---

## 등급 시스템

| 용어 | 설명 |
|------|------|
| **코스트** | 유닛 구매 가격 (1~5코스트) |
| **별 등급/성** | ★(1성) → ★★(2성) → ★★★(3성) |
| **합성** | 동일 유닛 3개 → 상위 등급으로 업그레이드 |

## 종족 (Race) - 3종

| 종족 | 아이콘 | 특성 |
|------|--------|------|
| **키키** | 🐱 | 회피, 이동속도 |
| **벨루아** | 💜 | 흡혈, 적 약화 |
| **엘프** | 🧝 | 마나 회복, 스킬 위력 |

## 무기 (Weapon) - 5종

| 무기 | 아이콘 | 특성 |
|------|--------|------|
| **대검** | 🗡️ | 방어력, 체력, 탱킹 |
| **지팡이** | 🪄 | 스킬 위력, 마법 피해 |
| **활** | 🏹 | 공격속도, 사거리 |
| **쌍검** | ⚔️ | 치명타, 암살 |

---

## 시너지 관련

| 용어 | 설명 |
|------|------|
| **시너지** | 같은 종족/무기 유닛 조합 시 발동되는 보너스 |
| **티어** | 시너지 발동 단계 (2/3/4명 등) |
| **활성 시너지** | 현재 발동 중인 시너지 |

---

## 전투 관련

| 용어 | 설명 |
|------|------|
| **라운드** | 전투 단위 (총 10라운드) |
| **웨이브** | 해당 라운드의 적 구성 |
| **준비 페이즈** | 유닛 배치/구매 시간 (30초) |
| **전투 페이즈** | 자동 전투 진행 |
| **스킬** | 마나가 차면 자동 발동되는 특수 능력 |

---

## 효과 관련

| 용어 | 영어 | 설명 |
|------|------|------|
| **회피율** | Evasion | 공격 회피 확률 |
| **흡혈** | Lifesteal | 피해량의 일부 회복 |
| **치명타** | Critical Hit | 추가 피해 발생 확률 |
| **스턴** | Stun | 행동 불가 상태 |
| **버프** | Buff | 긍정적 효과 |
| **디버프** | Debuff | 부정적 효과 |

---

## 코드 용어 (개발자용)

| 용어 | 설명 |
|------|------|
| **unitId** | 유닛 타입 식별자 (예: 'kiki_guardian') |
| **id** | 유닛 인스턴스 고유 ID (UUID) |
| **position** | 보드 상 위치 `{row, col}` |
| **initialPositions** | 전투 시작 시 저장된 유닛 위치 (복구용) |
| **cells** | 보드의 2D 배열 `cells[row][col]` |
| **tick** | 전투 업데이트 주기 (100ms) |
